# v0.1.1 Plan

## Goal

Ship a small but practical stability release after the initial public launch.

## Priority 1

- Improve onboarding responses so agents get more explicit next-step guidance
- Add more precise conflict messages for overlapping path claims
- Improve README examples for Claude Code and Codex setup
- Add a startup flow example that shows team creation and team join behavior

## Priority 2

- Add tests for path overlap edge cases
- Add tests for task claim override behavior
- Add tests for scope validation against real Git changes
- Improve release and publishing documentation

## Priority 3

- Add optional workspace summary tool for dashboards or status views
- Add exportable board snapshot JSON
- Add better stale-session diagnostics

## Non-Goals

- No web dashboard yet
- No hosted backend
- No autonomous orchestration engine
- No semantic merge system

## Recommended Versioning

- `v0.1.1`
  - documentation polish
  - onboarding improvements
  - validation and conflict messaging improvements

- `v0.2.0`
  - handoff flow
  - boundary warnings
  - worktree helpers
  - dashboard work
